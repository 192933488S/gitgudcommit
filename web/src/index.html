libgit2 is started as a webworker. Open web console to test it (Right-click + inspect in Google Chrome).

In the console select git_worker.js to receive commands.

Start with typing the command: 

jsgitinit()

Then try cloning using: 

jsgitclone("https://somehost/path-to-gitrepository.git","destinationfolder")

Finally list the contents of the repository using 

FS.readdir("destinationfolder")
